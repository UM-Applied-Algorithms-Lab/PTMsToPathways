# Pathways List Example

A named list with names of Pathways and string vectors that hold all the
gene symbols from those pathways

## Usage

``` r
ex.pathways.list
```

## Format

List of 1657 elements:

- pathway:

  A pathway name with the list of genes involved

## Source

"created from a Bioplanet.csv file in PathwayCrosstalkNetwork"

## Examples

``` r
head(ex.pathways.list)
#> Error: object 'ex.pathways.list' not found
```
