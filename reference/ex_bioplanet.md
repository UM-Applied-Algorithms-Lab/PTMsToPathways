# Bioplanet

A database file from the bioplanet website

## Usage

``` r
ex_bioplanet
```

## Format

Dataframe of 196 rows and 4 columns

- PATHWAY_ID:

  The pathway a gene belongs to

- PATHWAY_NAME:

  The name of the pathway a gene belongs to

- GENE_ID:

  A unique gene ID

- GENE_SYMBOL:

  An abrivation for the name of the gene

## Source

"https://www.frontiersin.org/journals/pharmacology/articles/10.3389/fphar.2019.00445/full"
