# Correlation Matrix

Matrix relating the correlation values between ptms based on Spearman
Similarity.

## Usage

``` r
data(ex_ptm_correlation_matrix)
```

## Format

Square matrix of 908 rows and columns:

- rows and columns:

  Ptm names

- values:

  Strength of correlation
