# Symmetric set difference of two vectors

Returns elements that are in either `x` or `y` but not in both (i.e.,
the outer- or exclusive union).

## Usage

``` r
outersect(x, y)
```

## Arguments

- x:

  A vector.

- y:

  A vector.

## Value

A sorted vector of elements present in exactly one of `x` or `y`.
